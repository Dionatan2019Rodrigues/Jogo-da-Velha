# Jogo da Velha assembly

## Descrição

Jogo da velha desenvolvido em lingaguem assembly, no qual o usuário joga contra a máquina. Sendo necessário do programa Mars4_5.jar para ser executado.

- `$ java -jar Mars4_5.jar`
